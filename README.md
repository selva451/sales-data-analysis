# Sales Data Analysis Dashboard

## Overview

This project is a Sales Data Analysis Dashboard developed using Django and Python. The dashboard helps analyze sales data, generate reports, and visualize business performance.

## Features

* Sales data management
* Dashboard analytics
* Data visualization
* Report generation
* Database integration
* User-friendly interface

## Technologies Used

* Python
* Django
* HTML
* CSS
* SQLite

## Project Structure

sales_dashboard_project/
├── manage.py
├── sales_dashboard/
├── templates/
├── static/
└── database files

## How to Run

1. Clone the repository
2. Install dependencies
3. Run migrations
4. Start the server

```bash
python manage.py runserver
```

5. Open browser:

```bash
http://127.0.0.1:8000/
```
